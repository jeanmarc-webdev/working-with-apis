# Working with APIs - Python Project

This project was built while following *Python Crash Course* by Eric Matthes.

It focuses on working with APIs in Python, processing JSON data, and visualizing results using Plotly and Matplotlib.

---

## Project Overview

This project demonstrates how to:

- Make API requests using Python
- Process JSON responses
- Work with GitHub API data
- Monitor API rate limits
- Extract and analyze repository information
- Visualize data using Plotly

---

## Technologies Used

- Python 3.x
- requests
- pandas
- matplotlib
- plotly

---
